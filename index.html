Estrutura Básica do HTML

Nesta aula, o professor retoma o assunto da aula "Ferramentas Utilizadas" com mais detalhes. Ele apresenta a nova tag <title> e nos mostra o plugin Live Server para visualizar em tempo real como a página ficará na web.

Falando Sobre Tags

Nesta aula, o professor fala sobre as tags <html>, <body> e <head>, além de nos apresentar as tags <i>, <strong>, <input> e <img>. Vou explicar o que aprendi:

  • <html>: Define que o documento é HTML. É a tag "mãe".
  • <head>: Contém as configurações e o título da página (<title>).
  • <body>: É o corpo do site, onde fica tudo o que é visível.
  • <input>: Usada para interação e entrada de dados. É uma tag de autofechamento.
  • <img>: Essencial para exibir imagens usando o atributo src.
  • <strong> e <i>: Usadas para dar ênfase (negrito) e itálico ao texto.

Resumo da Aula

O professor reuniu a teoria e a prática, abordando pontos fundamentais como listas e links.

1. Listas Ordenadas e Não Ordenadas

As listas servem para agrupar itens de forma lógica:

  • Listas Ordenadas (<ol>): Usadas quando a ordem é importante. O navegador numera automaticamente.
    1. item A
    2. item B
    3. item C
  • Listas Não Ordenadas (<ul>): Identifica itens com marcadores (pontinhos).
    • item A
    • item B
    • item C

2. Links e Navegação

Para criar links, utilizamos a tag <a> com o atributo href. Também aprendemos sobre o atributo target:

  • target="_blank": Abre o link em uma nova aba.
  • target="_self": Abre na mesma aba (padrão).
